### **001: Single Agent Identity** `001_single_agent_identity.py`
**Core Concept:** Identity-driven behavior

**What you'll learn:**
- How to define agents by their identity rather than procedural steps
- The power of declarative agent design
- Why "what agents ARE" matters more than "what they DO"

**Key Insight:** Simple identity definitions lead to consistent, natural behavior patterns.

**Run:** `python examples/001_single_agent_identity.py`

## üöÄ Quick Start Guide

### Prerequisites
1. **Install Praval** (see main README for installation instructions)
2. **Set up API keys** for at least one LLM provider:
   ```bash
   export OPENAI_API_KEY=your_key_here
   # or
   export ANTHROPIC_API_KEY=your_key_here
   ```

### Running the Examples

**Start simple and progress through the sequence:**

```bash
# Begin with foundational concepts
python examples/001_single_agent_identity.py
python examples/002_agent_communication.py  
python examples/003_specialist_collaboration.py

# Explore architectural patterns
python examples/004_registry_discovery.py

python examples/006_resilient_agents.py

# Discover emergent intelligence
python examples/007_adaptive_agent_systems.py
python examples/008_self_organizing_networks.py
python examples/009_emergent_collective_intelligence.py
```

### Learning Tips

1. **Follow the sequence** - Each example builds on concepts from previous ones
2. **Read the code** - The implementations demonstrate best practices
3. **Experiment** - Modify the examples to explore different behaviors
4. **Think in agents** - Focus on what each agent IS rather than what it DOES
5. **Observe emergence** - Watch for behaviors that arise from agent interactions

## üåü The Praval Philosophy in Action

These examples embody Praval's core principles:

1. **Simplicity Over Complexity** - Each agent has a clear, focused purpose
2. **Specialization and Collaboration** - Agents excel individually and collectively  
3. **Identity-Driven Design** - Behavior emerges from what agents ARE
4. **Self-Organization** - No central control or hierarchy needed
5. **Emergent Intelligence** - The whole becomes greater than the sum of its parts

### From Simple to Sophisticated

The progression shows how:
- **Simple agents** (001-003) establish fundamental behaviors
- **Coordinated systems** (004-006) enable reliable, scalable operations
- **Emergent networks** (007-009) achieve true collective intelligence

### The Coral Reef Metaphor

Like a coral reef ecosystem:
- Each agent has a specialized role (like different coral species)
- Interactions create complex, emergent behaviors
- The system becomes more intelligent and capable over time
- Simple components combine to create beautiful, sophisticated wholes
